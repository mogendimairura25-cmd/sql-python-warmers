# 📊 Sales Dashboard (SQL + Python)

This project analyzes sales data using Python and SQL, generating insights and visualizations.

## 🚀 Features
- Data cleaning and aggregation using Pandas
- SQL queries for grouped analysis
- Bar chart visualization of product sales

## 🛠️ Tech Stack
- Python (Pandas, Matplotlib)
- SQL
- CSV dataset

## 📂 Project Structure

├── analysis.py
├── data.csv
├── queries.sql
└── README.md
